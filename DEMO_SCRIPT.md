# AEGIS — Demo Video Script

## Format
- Length: 90 seconds primary cut, plus a 30-second teaser.
- Voiceover: English. Tone: calm, confident, technical.
- Capture beforehand: the live dashboard (Overview, Strategy, Risk Sentinel tabs), a bscscan transaction page, and the GitHub Actions run list.

## The 90-second cut
0:00 - 0:10 — Hook
On screen: dashboard Overview, the LIVE - BSC indicator pulsing.
VO: Most trading agents optimize for returns. In a competition where the biggest drawdown gets you disqualified, AEGIS optimizes for survival first.

0:10 - 0:30 — Autonomy and identity
On screen: Strategy tab — regime TREND, CoinMarketCap Fear and Greed, signals, the reasoning log scrolling.
VO: AEGIS reads live CoinMarketCap signals — Fear and Greed, momentum, regime — and decides on its own. Every decision is logged and explainable. The agent carries a verifiable on-chain identity through the BNB AI Agent SDK, ERC-8004 agentId 1282.

0:30 - 0:55 — Money shot: the Risk Sentinel
On screen: Risk Sentinel tab — the verdict table; cursor hovers and the frame zooms on the KILL row.
VO: Here is what makes us different. A deterministic Risk Sentinel sits between the AI and the chain. A healthy trade is approved. Too volatile, and it resizes the position itself. A slippage spike or an unknown token is vetoed. And on a drawdown breach it does not slow down — it kills the trade and exits to stable. The edge is the guardrail.

0:55 - 1:15 — Real on mainnet
On screen: bscscan transaction 0x353002..., then a green GitHub Actions run named AEGIS Trade.
VO: This is not a mock. Live swaps on BNB Chain mainnet through Trust Wallet TWAK, executed hands-free every day by GitHub Actions. Here is a real transaction on bscscan.

1:15 - 1:30 — Close
On screen: AEGIS logo and the stack summary line.
VO: AEGIS — Autonomous Execution and Guarded Intelligence. Built on the full BNB stack: CoinMarketCap, Trust Wallet, and the BNB AI Agent SDK. Survival-first, fully transparent, live on mainnet.

## Shot list
1. Dashboard Overview with live equity and PnL tickers.
2. Strategy tab: regime, signals, reasoning log scrolling.
3. Risk Sentinel tab: verdict table with a slow zoom onto the KILL row.
4. bscscan transaction page for the sample swap.
5. GitHub Actions: the AEGIS Trade workflow showing a green run.
6. Logo end card with the one-line stack summary.

## 30-second teaser
On screen: cut straight to the Risk Sentinel verdict table and end on the KILL row.
VO: AEGIS is an autonomous trading agent on BNB Chain. An LLM proposes trades. A deterministic Risk Sentinel can approve, resize, veto, or kill them before they reach the chain. In a competition decided by drawdown, the edge is the guardrail. Live on mainnet, fully transparent, verifiable on-chain.

## Production notes
- Keep each static framing under five seconds; favor slow pans.
- Show real numbers on screen, never placeholders.
- Subtitle the voiceover for silent autoplay.
- The end card must show the repository and dashboard URLs.
